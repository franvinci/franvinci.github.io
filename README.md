# Francesco Vinci — Personal Website

Academic homepage for Francesco Vinci, Research Fellow at the University of Padua.

## About

AI and process science researcher focusing on data-driven and explainable process simulation models for operational decision-making. Research spans process mining, machine learning, and explainable AI, with applications in healthcare and industry.

## Tech

- Static HTML/CSS/JS
- Bootstrap
- Typed.js for the intro animation

## Run locally

```bash
# With Python
python -m http.server 8000

# Or with Node
npx serve
```

Open [http://localhost:8000](http://localhost:8000).

## Deploy

GitHub Pages: push to the `main` branch. The site is served from the repo root.
